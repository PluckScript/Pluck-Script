# Language Features

Pluck supports the following features:

- Variables
- Arithmetic operations (+, -, *, /, %, **)
- Comparison operators (>, <, >=, <=, ?=, !=)
- Logical operators (and, or, not)
- If, elif, else statements
- While loops
- For loops
- Lists (with @{{ ... }}@ syntax)
- List indexing and mutation
- User-defined functions
- Built-in functions (out, input, length, int, str, float)
- Comments (//)
- Return and break statements
- Importing modules
- Boolean, integer, float, and string types

See individual docs for details on each feature.
